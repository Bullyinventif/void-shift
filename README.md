## Void Shift

Runner réflexe en corridor néon. On contrôle un diamant qui doit changer
de voie (haut / bas) pour éviter les murs, en attrapant des power-ups.

**Contrôles**
- `↑` / `Espace` / `W` ou *tap en haut* → voie supérieure
- `↓` / `S` ou *tap en bas* → voie inférieure
- *Swipe* haut/bas sur mobile
- `Entrée` pour (re)lancer

## Déploiement (GitHub Pages)

1. Pousser ce dépôt sur GitHub.
2. Settings → Pages → Branch `main` → `/root`.
3. Le jeu sera servi à `https://<user>.github.io/<repo>/Void%20Shift/`.
